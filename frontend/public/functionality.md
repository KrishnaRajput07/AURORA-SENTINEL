# Public — Functionality

This directory contains static public assets served directly by the web server.

## Key Files

### `index.html`
- HTML template for the React SPA (Single Page Application)
- Root mount point (`<div id="root">`) for React rendering

### `logo.png`
- AURORA Sentinel logo image used in the app bar and branding

### `manifest.json`
- Web app manifest for PWA (Progressive Web App) configuration
