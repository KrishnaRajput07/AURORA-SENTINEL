# Setup Scripts — Functionality

PowerShell scripts for Windows environment setup and deployment.

## Scripts

### `deploy.ps1`
- Deployment automation script
- Handles build and deployment steps

### `quick_start.ps1`
- Quick start script for new developers
- Sets up the environment and starts services

### `setup_ai_local.ps1`
- Local AI setup script
- Configures Ollama and other local AI dependencies

### `unlock_gpu.ps1`
- GPU unlock/enablement script for Windows
- Helps resolve GPU access issues on Windows systems
